TAGS DE TEXTOS

Tags de título, parágrafo e formatação de texto

As tags de formatação de textos, não servem somente para formatar os textos de forma visual, o principal objetivo dessas tags é deixar o html semântico, ou seja, descrever o conteúdo do documento para o browser para o melhor entendimento. Algumas das tags abaixo já foram vistas no capítulo "Semântica", mas serão abordadas novamente para facilitar o entendimento.

Tag de título

Existem 6 tipos de títulos diferentes em html classificados pelo tamanho de fonte que vão desde <h1></h1> até <h6> </h6>.

<h1> </h1>

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> </h5>

<h6> </h6>

Motores de busca utilizam os títulos, ou seja, as tags de título para indexar a estrutura e a organização do conteúdo dos sites.

Essa estrutura indexada segue o padrão: <h1> para os títulos principais, <h2> para os subtítulos, <h3> para os subtítulos dos subtítulos e contínua seguindo essa ordem do mais importante ao menos importante.

Dica: Use as tags de título somenta para títulos, jamais pela formatação da fonte!

Tag de parágrafo

A tag que representa um parágrafo dentro do html é a tag <p></p>.

<p> Páragrafo aqui! </ p>

Observação: A tag de parágrafo remove todos os espaços adicionais, por isso para colocar espaços em brancos propositalmente, não adianta apenas utilizar espaço, você deve ou criar um estilo para adicionar o espaço ou utilizar a entidade   do html.

Quebra de linha

Para quebrar linhas utilizamos a tag </br>.

Existe também a tag <hr></hr> que em adicional a quebra de linha, ela cria um desenho de uma linha horizontal indicando o fim da seção.

Tags de formatação de textos

A tag <b></b> é utilizado para deixar o texto em negrito, apenas por questão de estilização.

A tag <strong></strong> tem o mesmo efeito que a tag b visualmente, mas ela serve espeficicamente para ênfatizar o conteúdo do texto que está dentro dela.

A tag <i></i> serve para deixar o texto em itálico, não tem efeito algum na semântica.

Outra tag que deixa o texto em itálico é a tag <em></em>, mas diferente da tag i ela serve vai além de deixar apenas o texto em itálico, ela serve também para dar ênfase ao texto.

A tag <ins></ins> mostra que um texto foi inserido na página.

Em conjunto com a tag <cite></cite> é utilizado a tag <blockquote></blockquote> ou a tag <q></q> para inserir citações, a tag <blockquote></blockquote> para citações maiores e a tag <q></q> para citações menores.

A tag <abbr></abbr> apresenta abreviações, juntamente com a forma completa.

A tag <address></address> serve para especificar as informações de contato do autor/proprietário de um documento ou artigo

Essa é uma tag de definição <dfn></dfn>, ela serve para descrever algum conteúdo, por exemplo, no contexto de um dicionário você usaria da seguinte forma <dfn>Pneumoultramicroscopicossilicovulcanoconiótico</dfn> Indica alguém ou algo relacionado com uma doença aguda nos pulmões que se origina com a inalação de cinzas vulcânicas.

A tag <code></code> serve para apresentar trechos de códigos dentro de um parágrafo.

A tag <sub></sub> serve para deixar um texto subscrito. Exemplo: H 2O = Água

A tag <sup></sup> serve para deixar um texto sobscrito. Exemplo 2 2 = 4

A tag <small></small> reduz o tamanho do texto, o objetivo é referir informações redundantes ou inválidas.
