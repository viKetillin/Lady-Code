Listas do HTML

Lista é um importante recurso de HTML, pois permite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. São recursos extremamente usados, inclusive quando nem imaginamos que ele esta sendo usado, como no caso de menus. Hoje, boa parte dos menus em HTML é feito com listas. Há três tipos de listas:

  • Não ordenada
  • Ordenada
  • De definição

Listas Ordenadas

As listas ordenadas ou numeradas são usadas para indicar alguma sequência ou numeração As tags usadas para criar uma lista não ordenada são <ol> e <li>. A tag <ol> é usada para definir a lista e a tag <li> é usada para cada item da lista.

O <header>; é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos de <h1> a <h6>, até elementos para representar imagens, parágrafos ou mesmo listas de navegação.

<html>

<head>

<title>Listas em HTML</title>

<head>

<body>

<p>Listas de Aprovados</p>

<ol>

<li>Bia</li>

<li>Pedro</li>

<li>João</li>

</ol>

</body>

</html>

Resultado

Listas de Aprovados

  1. Bia
  2. Pedro
  3. João
Listas Não Ordenadas

As Listas Não Ordenadas são iniciadas com a tag <ul> e são representadas por pequenos “bullets”.

<html>

<head>

<title>Listas em HTML</title>

<head>

<body>

<p>Lista de Compras</p>

<ul>

<li>Queijo</li>

<li>Leite</li>

<li>Arroz</li>

<li>Pão</li>

</ul>

</body>

</html>

Resultado

Lista de Compras

  • Queijo
  • Leite
  • Arroz
  • Pão
Listas de Definição

Listas de definição de definição são usadas em assuntos onde há um termo a em sequência a sua definição, estilo perguntas e respostas. As tags usadas são <dl>, <dt> e <dd>.

  • <dl> definition list - cria a lista de definição
  • <dt> difinition term - onde o termo é proposto
  • <dd> definition definition - onde o termo é definido

<html>

<head>

<title>Listas em HTML</title>

<head>

<body>

<p>Principais termos do glossário de Cloud Computing</p>

<dl>

<dt>BaaS</dt>

<dd>Backend Como Serviço</dd>

<dt>IaaS</dt>

<dd>Infraestrutura Como Serviço</dd>

<dt>PaaS</dt>

<dd>Plataforma Como Serviço</dd>

<dl>SaaS</li>

<dd>Software Como Serviço</

</dl>

</body>

</html>

Resultado

Principais termos do glossário de Cloud Computing

BaaS
Backend Como Serviço
IaaS
Infraestrutura Como Serviço
PaaS
Plataforma Como Serviço
SaaS
